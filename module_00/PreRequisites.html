Skills, Technology and Resources

Required Skills:

In order to be successful in this course, students must have a basic understanding of keyboarding, navigating the internet and possess good communication skills.

Technology:

For this course, you will be expected to use several internet websites, some of which have minimum system requirements. Please check your computer to make sure that it meets these requirements.

Required Resources:
